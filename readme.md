# Cafea（架空カフェサイト）

Cafeaは、架空のカフェをイメージして制作したWebサイトです。  
店内の雰囲気やメニューの魅力が伝わるよう、シンプルかつ温かみのあるデザインを意識しています。

## 使用技術

- HTML / CSS / JavaScript
- Swiper.js（スマホのメニューセクションをフェードで切り替え）
- Git によるバージョン管理
- VS Code 開発環境

## 主な機能・こだわりポイント

- レスポンシブ対応（PC／スマホ）
- スマホではメニュー上の画像をSwiperで1枚ずつ表示（フェード切り替え）
- メニュー項目はスマホでトグル表示（クリックで開閉）
- 全体の配色・フォントに統一感を持たせた温かみのあるデザイン
- Gitによる履歴管理
